Set up a new project

Please enter a name for your project - this will act as the name used for your website and also as the name used when entering into your phone to start collecting data.

Once you have created a project you can then design a form for data collection. You can also amend the look and feel of your homepage by adding some explanatory text about your project and upload an image.

Please read the instructions prior to proceeding.

Project names must only contain letters, numbers and underscores.

Or

Alternatively, upload an existing EpiCollect project definition (advanced users, see instructions)


Continue

Now you can specify whether you would like your project to be public (i.e. available for anyone to view data via the web) or private (will require a user to login to see data).

You can add users to a private project using the Managers and/or Curators tab on the Manage Project screen.


Continue Back

Here you can choose whether you would like your project to appear in a list of projects available for anyone to browse.

This will encourage users to add data to your project. If you choose to do this then consider adding a description and project image through the Manage Project page.

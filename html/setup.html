Enter Database Settings

Please fill out the form below to give EpiCollect access to your MySQL database. This form will only work once, after the initial settings have been written the settings file must be edited directly. All the below credentials will be written to a file in an encrypted format.

MySQL Server

The hostname or IP address of the MySQL server you wish to store your EpiCollect data on.

Database Name

The name of the database instance within MySQL that will store the epicollect data.

User Name

The username used to access the EpiCollect database (should have SELECT, UPDATE, INSERT and DELETE rights to the database)

Password

The password that is associated with username
